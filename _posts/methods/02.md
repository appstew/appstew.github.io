- 출처 : https://blog.outsider.ne.kr/271

```java
String[] temp = { "aa", "bb", "cc" };
for (int i = 0; i < temp .length; i++) { 
    System.out.println(temp[i]);
}
```
- 위 코드는 아래처럼 향상된 for구문으로도 쓸 수 있다.
```java
String[] temp = { "aa", "bb", "cc" };
for (String el : temp) {
    System.out.println(el);
}
```
